To deploy application on test follow below steps.
1. Changes in SQApp.js located on /SmartQuote/SQ/SQApp.js
2. Changes in global.properties located on /SmartQuote/WEB-INF/src/global.properties
3. Changes in context.xml located on /SmartQuote/META-INF/context.xml
4. Changes in log4j2.properties located on /SmartQuote/WEB-INF/src/log4j2.properties
